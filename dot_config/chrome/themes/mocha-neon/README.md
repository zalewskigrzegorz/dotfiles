# Mocha Neon — Chrome / Comet theme

Unpacked Chromium theme extension. Forked from Dracula PRO; all hex bumped to
Mocha Neon palette (`docs/mocha-neon-palette.md`).

## Install (Chrome / Comet / any Chromium browser)

1. Open `chrome://extensions` (or `comet://extensions`).
2. Enable **Developer mode** (top-right toggle).
3. Click **Load unpacked**.
4. Select this directory: `~/.config/chrome/themes/mocha-neon`.
5. Theme applies automatically.

## RGB → Mocha Neon mapping

| Token | Dracula PRO RGB | Mocha Neon RGB | Hex |
|---|---|---|---|
| base / frame | `34,33,44` | `30,30,46` | `#1E1E2E` |
| surface / toolbar | `69,65,88` | `69,71,90` | `#45475A` |
| text | `248,248,242` | `240,240,255` | `#F0F0FF` |
| overlay (subtle) | `121,112,169` | `127,132,156` | `#7F849C` |
| primary accent | `149,128,255` | `179,71,255` | `#B347FF` (mauve) |
